5. Deskripsi Website:
Situs web ini adalah Kolalimbah. Situs web ini tentang edukasi untuk pengelolaan limbah elektronik (electronic waste atau e-waste), dengan informasi yang menarik. bentuk informasinya yaitu artikel dan video. Situs web ini dibuat oleh kelompok kami bernama UdangKeju96 dari Universitas Majalengka yang tujuannnya untuk memberi edukasi tentang lingkungan untuk masyarakat yang luas. Situs web ini terbuka untuk publik yang besar.
6. Cara Instalasi & Menjalankan Proyek:
- Download proyeknya (dalam bentuk ZIP) dari repository ini
- Setelah berhasil download, UNZIP atau ekstrak file ZIP proyeknya
- Setelah itu, jalankan proyek ini di Visual Studio Code
- Setelah itu, jalankan proyek ini dengan Start Debugging file HTML atau menggunakan fitur ekstensi VSCODE Live Server di file HTML
